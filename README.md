# موقعي الشخصي | My Personal Portfolio

موقع ويب تعريفي شخصي بسيط ومصمم بشكل عصري ومستجيب (Responsive) ليتناسب مع جميع الشاشات. تم بناؤه كخطوة أولى لتعلم وتطبيق أساسيات تطوير الويب ورفع المشاريع على منصة GitHub.

## 🚀 التقنيات المستخدمة
* **HTML5:** لبناء الهيكل الأساسي للموقع وتقسيم الصفحات.
* **CSS3:** لتنسيق الألوان، الخطوط، وتصميم واجهة مستخدم مريحة للعين.

## 📂 محتويات المشروع
* `index.html`: الملف الرئيسي الذي يحتوي على النصوص وأقسام الموقع (المقدمة، من أنا، التذييل).
* `style.css`: الملف المسؤول عن التصميم الجمالي والتنسيقات الخاصة بالموقع.

## 🛠️ كيف تشغل المشروع؟
1. قم بتحميل الملفات أو عمل `Clone` للمستودع.
2. افتح ملف `index.html` في أي متصفح إنترنت (مثل Chrome أو Edge).
